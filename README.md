--Güven Emlak - Gayrimenkul Yönetim Sistemi--
Bu uygulama, bir emlak ofisinin ilanlarını, müşteri bilgilerini ve satış/kira durumlarını dijital ortamda yönetebilmesi için C# WinForms ve SQL Server teknolojileriyle geliştirilmiştir.

-Öne Çıkan Özellikler-
SQL Server Entegrasyonu: SqlConnection ve SqlCommand sınıfları kullanılarak veritabanı ile eş zamanlı veri iletişimi sağlanmıştır.

Tam CRUD Desteği: Gayrimenkul kayıtlarının listelenmesi, yeni kayıt ekleme, mevcut kayıtları güncelleme ve silme fonksiyonları mevcuttur.

Dinamik UI Yönetimi: Seçilen site adına (Çiçek, Kardelen, Gül, Papatya) göre arayüz renklerinin otomatik değişmesi gibi interaktif görsel geri bildirimler içerir.

Detaylı Takip: Gayrimenkul bilgileri (oda sayısı, metrekare, fiyat, blok/no) ile müşteri iletişim bilgilerinin entegre yönetimi.

Güvenli Giriş: Yetkili kullanıcı adı ve şifre kontrolü ile korunan giriş paneli.

-Teknolojiler-
Programlama Dili: C#

Veritabanı Mimari: ADO.NET (SQL Client)

Veritabanı Yönetimi: Microsoft SQL Server

Arayüz: Windows Forms (WinForms)
